# When to skip Loom and just send text

## Context
From [[Loom Playbook]].

## Content

- If their existing site is already decent — Loom critiquing a good site looks petty
- If you've already sent them a text version (don't double-touch unless 14+ days passed)
- If you're sending more than ~10 in a day (Loom doesn't scale; switch to text-batch)
