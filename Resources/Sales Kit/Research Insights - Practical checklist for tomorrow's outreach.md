# Practical checklist for tomorrow's outreach

## Context
From [[Research Insights]].

## Content

- [ ] Subject line 2-4 words, lowercase, references them
- [ ] 75-100 word body, 6-8 sentences, single CTA
- [ ] Inline mockup image (no Loom for v1, save for v2)
- [ ] Send Tue/Wed/Thu 6:30am OR 7pm PT
- [ ] First line references something specific to them
- [ ] No subject line spam-words ("Free", "ASAP", "Limited")
- [ ] Suppression list checked before each send
- [ ] Daily cap of 30-40
- [ ] Track reply rate per subject variant
