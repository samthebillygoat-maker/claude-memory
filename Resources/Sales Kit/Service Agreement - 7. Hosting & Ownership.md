# 7. Hosting & Ownership

## Context
From [[Service Agreement]].

## Content

- Site is deployed on Cloudflare Pages under Client's domain.
- Domain is owned by Client; Designer may purchase on behalf if requested ($10-20 pass-through cost).
- After final payment, Client owns the website code and design. Designer may include the project in a portfolio (with name/photos blurred if Client requests).
