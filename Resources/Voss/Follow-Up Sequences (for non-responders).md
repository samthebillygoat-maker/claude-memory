# Follow-Up Sequences (for non-responders)

Use these only when there has been zero reply to the initial outreach. Stop sending after Day 30. No exceptions.

### Day 5 — Accusation Audit

> Subject: probably the wrong time
>
> Hey {{landlord_name}} —
>
> You are probably thinking this is another arbitrage pitch, that you have already heard this one before, and that 12-month commitments from strangers on the internet are too good to be true.
>
> All fair.
>
> The only thing I want to know is whether {{address}} is still available, or whether I should close the file. Two-word reply works.
>
> Sam

### Day 10 — Calibrated Question

> Subject: {{address}}
>
> {{landlord_name}} —
>
> It seems like the listing might already be filled, or the timing is off on your end. Either is fine.
>
> What would need to be true for a guaranteed 12-month tenant on {{address}} to be worth 10 minutes on the phone?
>
> If the answer is nothing, that is the answer. No hard feelings.
>
> Sam

### Day 30 — No-Oriented Close-Out

> Subject: closing the file on {{address}}
>
> {{landlord_name}} —
>
> Have you given up on finding a 12-month tenant for {{address}}, or is it still on the market?
>
> I am closing my file on this one today either way. If the situation changes in the next few months, my number is below — text me directly and you skip the queue.
>
> Appreciate the time, even if the answer was no.
>
> Sam
