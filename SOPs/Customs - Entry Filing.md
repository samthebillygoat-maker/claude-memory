# 📋 SOP — Entry Filing Process

---

## Pre-Filing Checklist

- [ ] Commercial invoice received from client
- [ ] Packing list received
- [ ] Bill of lading / air waybill received
- [ ] HTS code confirmed for each commodity
- [ ] Country of origin verified
- [ ] Tariff exposure calculated:
  - Section 232 (metals, semis, pharma, autos)?
  - Section 122 (15% global surcharge — USMCA and S232 goods exempt)?
  - Section 301 (China + 15 economies)?
  - AD/CVD applicable?
- [ ] FDA prior notice filed (if food/drug/cosmetic)?
- [ ] USDA/APHIS permit in hand (if plants/animals)?
- [ ] NOAA Certificate of Admissibility (if seafood)?
- [ ] FSMA certification (Indonesian shrimp/spices from Java/Lampung)?
- [ ] CBP Forced Labor risk checked (WRO list, UFLPA entities)?
- [ ] Import Alert check (FDA IA 45-02, 99-08, 99-48)?
- [ ] Transshipment risk assessed?

---

## Filing

- [ ] Enter in ACE (Automated Commercial Environment)
- [ ] Select correct entry type (formal/informal/TIB/etc.)
- [ ] Confirm IOR matches POA on file
- [ ] Apply applicable duty rate reductions (USMCA, GSP, etc.)
- [ ] Note: USMCA goods exempt from Section 122 but NOT from Section 301
- [ ] Submit entry
- [ ] Pay estimated duties via ACH (duty payment ACH — separate from refund ACH)

---

## Post-Filing

- [ ] Save entry summary number (needed for CAPE refund filing if IEEPA duties paid)
- [ ] Monitor for CBP exam or hold
- [ ] Notify client of any exams or holds immediately
- [ ] Update client file with entry details

---

## If Shipment is Detained

- [ ] Determine reason (WRO, UFLPA, import alert, admissibility issue)
- [ ] File review via CBP Forced Labor Portal (cbp.gov/trade/forced-labor) if forced labor related
- [ ] Coordinate with client on documentation
- [ ] Track 30-day response deadline

---

## CAPE Refund — If IEEPA Duties Were Paid

- [ ] Confirm entry is eligible (not AD/CVD, not reconciliation, no drawback, no open protests)
- [ ] Confirm IOR is enrolled with ACH refund authorization in ACE (separate tab)
- [ ] File CAPE Declaration via ACE Portal (max 9,999 entries per CSV)
- [ ] Note: cannot amend once accepted — double-check before submitting
- [ ] Processing time: 60–90 days after acceptance
