# 🔑 SOP — Guest Check-In

---

## 24 Hours Before Arrival

- [ ] Confirm booking details (dates, guest count, special requests)
- [ ] Verify cleaning is scheduled for checkout day before
- [ ] Send check-in message with:
  - Exact address
  - Key/lockbox code
  - Parking instructions
  - WiFi name and password
  - House rules reminder
  - Your contact number for emergencies

---

## Day of Check-In

- [ ] Confirm unit is clean and ready by 3pm
- [ ] Verify all supplies stocked:
  - Toilet paper (2 rolls per bathroom)
  - Paper towels (1 roll)
  - Dish soap, sponge
  - Shampoo, conditioner, body wash
  - Hand soap
  - Coffee/tea (if provided)
- [ ] Locks working — lockbox code active
- [ ] AC set to 74°F
- [ ] All lights working
- [ ] TV and WiFi working
- [ ] Welcome note on counter (optional but gets 5-star reviews)

---

## After Check-In (Within 2 Hours)

- [ ] Send follow-up message: "Hope you're settled in! Let me know if you need anything."
- [ ] Monitor Airbnb messages for any issues

---

## House Rules to Include in Listing

- No parties or events
- No smoking inside
- No additional guests beyond booked number
- Quiet hours 10pm–8am (Miami-Dade occupancy cap: max 2/bedroom + 2, absolute max 12)
- Trash must be taken out on [collection days]
