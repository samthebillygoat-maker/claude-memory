# ✅ SOP — Monthly Compliance Review

**Frequency:** First Monday of every month

---

## Regulatory Updates Check

- [ ] Check CBP CSMS bulletins for new guidance
- [ ] Check FDA import alerts for new additions/removals
- [ ] Check CBP WRO list for new Withhold Release Orders
- [ ] Check UFLPA entity list for updates
- [ ] Check Section 122 court case status (Euro-Notions Florida v. CBP — ruling could suspend collection)
- [ ] Check Section 301 investigation progress (USTR — July 24 remedy target)
- [ ] Check NOAA MMPA swimming crab stay status
- [ ] Check APHIS bulletin for plant/animal import rule changes

---

## Client Portfolio Review

- [ ] Any clients importing from newly sanctioned countries/entities?
- [ ] Any clients with commodities newly added to import alerts?
- [ ] Any clients with CAPE refund filings — check status
- [ ] Any clients with open protests or exam holds — follow up
- [ ] Any clients who haven't set up ACH refund enrollment in ACE?

---

## Tariff Rate Verification

- [ ] Section 122: Still 15%? Still in effect? (Expires Jul 24, 2026 or earlier if court rules)
- [ ] Section 232 metals: Annex I-A 50%, I-B 25% — any new inclusions?
- [ ] Section 232 auto parts: New quarterly window July 1–14, 2026 — flag for auto clients
- [ ] Section 301: Any new tariff actions from USTR?
- [ ] Ecuador exemption still in force?

---

## Internal Housekeeping

- [ ] All client POAs current and on file
- [ ] All client ACE access up to date
- [ ] Entry records filed and organized
- [ ] Invoices sent and collected
