# Tampa Showing Checklist — run this at EVERY unit

Take ~10 min per unit. Don't sell — ask their questions, take notes, take photos.

## 1. Connectivity (do FIRST, takes 60 sec)
- [ ] Speedtest app — record Mbps down / up / ping
- [ ] AT&T bars (Sam's carrier) — note in unit + at street
- [ ] Walk to farthest corner of unit — still 4+ bars?
- [ ] Note Wi-Fi network names visible (neighbor density signal)

## 2. Parking
- [ ] Assigned spot? Numbered?
- [ ] Guest parking — how many spots, overnight allowed?
- [ ] Street parking — any "permit only" or "2-hr" signs?
- [ ] Tow company sign posted? (red flag for strict enforcement)

## 3. Building shell
- [ ] Roof age — ask, or eyeball (curled shingles = old)
- [ ] Exterior paint condition
- [ ] Window age — single pane? sealed?
- [ ] Common-area smell — mildew? cigarette? pet?
- [ ] Mailbox area — orderly or trashed?
- [ ] Fire alarm panel — when last serviced sticker?

## 4. Unit interior
- [ ] **Water pressure** — run kitchen + bath faucets full blast
- [ ] **AC unit** — age sticker on outdoor condenser? Last service date?
- [ ] **Water heater** — age sticker, electric or gas, tank size
- [ ] **Kitchen vent hood** — does the fan actually work?
- [ ] **Laundry** — in-unit hookup, in-unit stack, or shared?
- [ ] **Outlets per bedroom** — count. <4 = problem for guest setups.
- [ ] **Smoke/CO detectors** — present in every required room?
- [ ] **Window AC vs central** — if window, count units needed

## 5. STR-fit red flags (MOST IMPORTANT)
- [ ] Any "No Short-Term Rental" or "Residents Only" signage?
- [ ] Security cameras pointed at entries? (HOA surveillance)
- [ ] Neighbor unit signs / decals (HOA culture)
- [ ] "Tenants only" parking enforcement signs?
- [ ] Lockbox already on door = good sign (rotation common here)
- [ ] Numbered keycode pads on neighbor doors = STR-friendly

## 6. Photos (8–12 per unit)
- [ ] Living room — wide angle from doorway
- [ ] Kitchen — wide + countertop close-up
- [ ] Master bedroom — wide + closet
- [ ] Second bedroom (or den)
- [ ] Bathroom — wide + shower
- [ ] Best view from any window
- [ ] Exterior — building front
- [ ] Parking spot / entry path
- [ ] Any amenity (pool, gym, laundry)

## 7. Neighbor signal (golden question)
- [ ] Knock on ONE neighbor door — if someone home, ask:
  - **"How's the noise here?"** (single most predictive question)
  - Follow with: "How long have you lived here?"
- [ ] If nobody home, note door decor — kids? dogs? older folks?

## 8. Walk the block (2 min)
- [ ] Any obvious complaint-magnet neighbor signs?
- [ ] Bars / restaurants within 1 block? (noise complaints come FROM not TO)
- [ ] Sidewalks lit?
- [ ] Anything that screams "this neighborhood files complaints"?

## 9. Owner-direct questions to ask THEM
- [ ] How long has unit been vacant?
- [ ] What did last tenant pay?
- [ ] Why did last tenant leave?
- [ ] Open to 12-month with corporate guarantor?
- [ ] Any restrictions on the lease re: guests / professional use?
- [ ] HOA or condo board involved? (kills deals)

## 10. After every tour — log immediately
- [ ] 1–5 score on STR fit
- [ ] Walk-away or counter?
- [ ] Note in vault as `Properties/[address].md`
