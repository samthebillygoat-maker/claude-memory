## Execution Order

1. **Generate all 7 in ChatGPT first.** Don't print anything until you've seen and approved all 7 — palette consistency matters across the set.
2. **Save each as PNG at max resolution.** Right-click → Save image → name them `tara_p1_sofa_center.png` etc.
3. **Order Piece 1 from Mpix** first (3-day ship), upload PNG, choose Standard Matte, no border.
4. **Upload Pieces 2-7 to Walmart Photo** (matte poster, not glossy), select same-day pickup at your local Walmart.
5. **Order Americanflat light oak frames** (Amazon) — buy 2× of each size in one order.
6. **Pick up Walmart prints**, inspect for banding/color shift before leaving the counter. Reprint anything that looks off.
7. **Wait for Mpix Piece 1 to arrive**, then hang in this order: Piece 1 first → flank with 2 + 3 → bedroom diptych → dining → bath.

## Troubleshooting Prompts

If ChatGPT output looks too vector/clipart → add to prompt:
> "thick visible brush strokes, oil pastel texture, hand-painted imperfection, color bleeding at edges, visible canvas weave, gouache layering, no clean vector edges anywhere."

If colors drift cool/muddy → add:
> "warm-leaning color temperature throughout, no cool gray undertones, no blue contamination in the olive."

If terracotta dominates → add:
> "terracotta used only as a small accent occupying no more than 8 percent of the canvas, olive and cream must visually lead."

Regen up to 3× per piece. If still off, swap shape language (arches ↔ dunes ↔ vessels) instead of fighting the palette.
